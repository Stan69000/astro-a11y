# Support

## Maintainer Model

`astro-a11y` is maintained by a single person.

## What to Expect

- Best effort support
- No guaranteed SLA
- Priorities: security, regressions, and usability blockers

## Best Way to Get Help

- Open a GitHub issue with reproduction steps
- Share exact command, output, and environment (Node + pnpm versions)
- For security concerns, follow [SECURITY.md](./SECURITY.md)
